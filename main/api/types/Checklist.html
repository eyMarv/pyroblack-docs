    Checklist¶

    class pyrogram.types.Checklist¶

    Describes a checklist.

    Parameters:¶

    • title (``str``) – Title of the checklist.

    • title_entities (List of :obj:`~pyrogram.types.MessageEntity`, *optional*) – Special entities that appear in the checklist title. May contain only Bold, Italic, Underline, Strikethrough, Spoiler, and CustomEmoji entities.

    • tasks (List of :obj:`~pyrogram.types.ChecklistTask`, *optional*) – List of tasks in the checklist.

    • others_can_add_tasks (``bool``, *optional*) – True, if users other than the creator of the list can add tasks to the list.

    • others_can_mark_tasks_as_done (``bool``, *optional*) – True, if users other than the creator of the list can mark tasks as done or not done.
